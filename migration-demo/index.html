Get started in the footbag community...
If you aren't already a member, please consider joining footbag.org. Membership is free, allowing posting to the forum, getting contact information for other footbag players, registering for competitions, etc. You also get to create a profile so other players can find you -- and if there aren't any clubs near you, you can create your own free club listing on our world-wide club list. Join today and upgrade to an annual IFPA membership for free, as well!
